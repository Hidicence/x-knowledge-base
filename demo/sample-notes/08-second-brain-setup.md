# Second Brain Setup 紀錄

## 我的工具選擇

最終選擇：Obsidian + Git backup + AI 輔助整理

為什麼不用 Notion：
- 沒有本地檔案，擔心資料主權
- 協作功能對個人用途過度設計
- Markdown export 不完整

為什麼選 Obsidian：
- 純 markdown 本地儲存
- Graph view 讓我看見知識連結
- Plugin 生態豐富（Dataview、Templater、Smart Connections）

## 我的資料夾結構

```
Second Brain/
├── Inbox/          # 所有新東西先放這裡
├── Notes/          # 永久筆記（atomic notes）
│   ├── Concepts/   # 概念、定義
│   ├── People/     # 關於特定人物的知識
│   └── Projects/   # 進行中的專案筆記
├── Literature/     # 讀書/文章筆記
├── Journal/        # 每日記錄
└── Archive/        # 不再活躍的內容
```

## 每日回顧習慣

- 早上：打開 Inbox，快速處理昨天堆積的東西（10 分鐘）
- 晚上：寫 daily note，記下當天學到什麼、決定什麼、有什麼問題

## 最大收穫

建立 second brain 一年後，最大的收穫不是「存了更多知識」，而是**思考更清晰**。
寫筆記的過程本身就是在整理思路。
