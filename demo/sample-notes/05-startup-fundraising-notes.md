# 早期融資筆記

## YC 面試準備整理

被問到最多的幾個問題：
1. 你在解決什麼問題？為什麼現在？
2. 你的目標客戶是誰？你怎麼找到他們的？
3. 你的 unfair advantage 是什麼？
4. 6 個月後你打算做什麼？

YC 最在意：**founder-market fit**（你為什麼是解決這個問題的正確人選？）

## Seed Round Checklist

- [ ] Deck：10-12 頁，包含 problem / solution / market / traction / team / ask
- [ ] Data room：財務預測、技術架構、法律文件
- [ ] Reference customers：3-5 個願意背書的早期用戶
- [ ] MRR 或 LOI：有任何收入比沒有好太多

## 幾個融資現實

- 大部分 VC 看 100 個 deal，投 1-2 個
- Email cold outreach 轉化率極低，warm intro 是關鍵
- Term sheet 出來前不要停止融資流程
- 評估 VC 不只看錢：他有沒有你的目標客戶的資源？

## 我的觀察

最近 AI 領域估值泡沫，但能清楚說明 defensibility 的公司仍然拿到好條款。
「我們在 GPT wrapper 上加了 UX」已經不夠了，需要：data moat、distribution moat、或 workflow integration。
